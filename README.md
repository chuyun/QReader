日常生活中，在电脑端一出现二维码，都要掏出手机扫码，然后将信息发给电脑，相当繁琐



该程序支持将二维码拖入程序中，返回二维码内容

下一版本，将会继续实现以下功能：

支持拖入网址、等其他信息，生成对应的二维码，并保存

支持将二维码内容发送到剪切板

支持输入信息，编码为二维码

可能会加入对条形码的支持

程序中使用ZXing的二维码解析库
